# Commands

```console
foo@bar:~$ hadoop fs -mkdir /input
foo@bar:~$ hadoop fs -mkdir /output
foo@bar:~$ hdfs dfs -ls /
foo@bar:~$ hdfs dfs -put example.txt /input
foo@bar:~$ hadoop jar example.jar /input/example.txt /output/example.txt
```
